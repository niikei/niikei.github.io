Created a Website!

I wanted a way to share what I'm doing, so I created a website. I chose GitHub Pages because it was free and seemed easy to use.
For "Currently Reading" page, I use Google Spreadsheet as a database and fetch data using GAS. You can sort the table by clicking on the column names. The table layout changes slightly between the desktop and mobile versions, which is a small detail I'm proud of. Try rotating your phone to see the difference.
The design is pretty basic, so I might improve it in the future.
